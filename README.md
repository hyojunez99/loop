# Infinite Image

## 프로젝트 한 줄 요약

라이브러리 없이 JavaScript로 무한 이미지 캐러셀을 구현한 UI 인터랙션 프로젝트로,  
이전·다음 버튼과 자동 재생·정지 기능을 포함한 슬라이드 동작을 직접 구현했습니다.

## 주요 기능

- 이전·다음 버튼을 통한 이미지 슬라이드 이동  
- 첫 이미지와 마지막 이미지가 자연스럽게 이어지는 무한 캐러셀 구조  
- 자동 재생 및 재생 정지 기능 구현  

---

## 기술 스택

- Frontend: HTML5, CSS3, JavaScript (ES6+)

## 배운 점 / 인사이트

### JavaScript UI 구현 경험

외부 라이브러리 없이도 JavaScript만으로 복잡한 UI 인터랙션을 구현할 수 있다는 자신감을 얻었습니다.

### DOM 구조 및 상태 관리 경험

슬라이드 UI에서는 DOM 구조 설계와 현재 위치 상태 관리가 핵심이라는 점을 이해하게 되었습니다.
